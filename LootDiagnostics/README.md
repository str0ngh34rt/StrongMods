# LootDiagnostics

Send loot drop information to the logs.

```
[LootDiagnostics] plagueSpitter dropped loot plagueLootConatiner
```

## Installation

* Copy the `LootDiagnostics/` directory into `Mods/`
* Make sure the `ModInfo.xml` appears one folder below `Mods/`, i.e. `Mods/LootDiagnostics/ModInfo.xml`, otherwise the
  mod won't be loaded
* Dedicated servers:
  * Server-side only
  * EAC-friendly
* All other deployments:
  * Deploy to host (in single-player this is your game)
  * EAC must be disabled
* There are no configuration options for now

# Progressive Biomes

Progressive Biomes rebalances wandering zombie spawns to make each biome feel like a meaningful step up in difficulty.

Instead of simply increasing every zombie equally, the mod separates biome-specific zombies—and Project Z bosses, when
installed into their own spawn groups so they can be tuned independently. This keeps special enemies and bosses rare
while dramatically increasing the number of ordinary infected roaming the world.

## Features

- Progressive biome difficulty from Pine Forest to Wasteland.
- Increased wandering zombie populations in every biome.
- Faster zombie respawn rates to keep the world feeling alive.
- Automatic support for Project Z, with bosses placed into their own spawn groups.
- Biome-specific zombies stay at similar spawn rates to vanilla.
- Does _not_ affect quest spawns or Blood Moon hordes—only ambient biome spawns.

Whether you're exploring, gathering resources, or traveling between traders, every biome becomes a more dangerous place,
with the Wasteland earning its reputation as the deadliest region in the game.

## Installation

* Copy the `ZZ_ProgressiveBiomes/` directory into `Mods/`
* Make sure the `ModInfo.xml` appears one folder below `Mods/`, i.e. `Mods/ZZ_ProgressiveBiomes/ModInfo.xml`, otherwise
  the mod won't be loaded

* Server-side only
* EAC friendly
* No configuration options for now
* Auto-detects Project Z (public or experimental)
